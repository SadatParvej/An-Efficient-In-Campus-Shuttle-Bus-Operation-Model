Required Software and Language:
	Python 3.9
	QGIS 3.30.2
	PyQt5

Instructions:

1. Go to "Map_folder", then "Projects" and double click the "Map_V" file. If you have QGIS installed this will automatically open in the QGIS application
2. It must be ensured that the python plugin is installed beforehand in QGIS, if it is already not installed. The "mapf.py" file can then be uploaded as a script through that plugin.
